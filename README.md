# sonjin1128.github.io

Personal academic homepage of Sungjin Bae, built with [Jekyll](https://jekyllrb.com/) and the [Minimal Mistakes](https://github.com/mmistakes/minimal-mistakes) theme, hosted on GitHub Pages.

## How to update

- **About page**: edit `index.md`
- **Research / Publications / CV**: edit files in `_pages/`
- **Profile photo**: replace `assets/images/profile.jpg`
- **Links & site settings**: edit `_config.yml`
- **Menu**: edit `_data/navigation.yml`

Changes pushed to `main` are automatically rebuilt and published at <https://sonjin1128.github.io> (allow 1–3 minutes).
